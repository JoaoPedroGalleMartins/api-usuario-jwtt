# API de Usuários com JWT

## 📌 Sobre o projeto
API REST desenvolvida em Java com Spring Boot para gerenciamento de usuários, com autenticação utilizando JWT.

---

## 🚀 Tecnologias
- Java
- Spring Boot
- PostgreSQL
- Spring Security
- JWT (JSON Web Token)
- JUnit e Mockito

---

## ⚙️ Funcionalidades
- Cadastro de usuários
- Login com geração de token JWT
- CRUD completo de usuários
- Validação de dados com Jakarta Validation
- Tratamento global de exceções
- Testes unitários de service

---
